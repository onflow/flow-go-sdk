## Installing

To start using the SDK, install Go 1.13 or above and run go get:

```sh
go get github.com/onflow/flow-go-sdk
```

## More Examples

Go here more [examples](https://github.com/onflow/flow-go-sdk/tree/master/examples) that use the SDK to interact with the [Flow Emulator](https://docs.onflow.org/emulator/).
